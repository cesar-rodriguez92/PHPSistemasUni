Definicion:
Un layout define la distribución general de los elementos html en un documento web. De manera general un layout divide el area de presentación del documento en secciones donde se mostrarán elementos de información relacionados: cabeceras, barras o menus de navegación contenido principal, contenido relacionado o secundario, pies etc. Inicialmente se utilizó el elemento tabla de html para definir la distribución de los elementos html en el documento, actualmente está practica está considerada obsoleta y se prefiere el uso de CSS para realizar el diseño del layout. Hasta html 4 se utilizaban los contenedores div o span para genrerar las zonas componentes de un layout, actualmente la version 5 de html incluye los elementos semanticos correspondientes para generar dichos elementos: header, nav, section, article, aside, footer, details summary. La forma principal de crear layouts utilizando CSS es usando la propieda float. Los layout más comunes se nombran por su estructura general: número de columnas (dos columnas, tres columnas), por la inclusión de una cabecera o pie: dos columnas cabecera y pie.
