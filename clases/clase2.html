Una table organiza la información en filas y columnas, cada fila está definida por la etiqueta (tr) (table row) y cada elemento de la columna por una celda, definida por la etiqueta (td) (table data) o (th) (table header)
Ejemplo:
Columna 1Columna 2Columna 3
Fila 1Dato 1Dato 2Dato 3
Fila 2Dato 4Dato 5Dato 6
Fila 3Dato 5Dato 8Dato 9Dato 10

Definicion:
De maner predeterminada una celda ocupa el ancho de una columna y el alto de una fila, podemos modificar este comportamiento con los atributos colspan y rowspan los cuales nos permiten definir que una celda ocupe más de una fila o más de una columna respectivamente, el valor del atributo debe ser numérico y entero indicará el número de filas o columnas que ocupará la celda.
Ejemplo:
12 (colspan)34
5678
910 (rowspan)1112

Definicion:
Es un elemento html que permite recolectar datos y enviarlos al servidor. estructura de un formulario.

Atributos

Action: Indica a donde sera enviado el formulario
Method: EL metodo mediante el cual es enviado(GET o POST)
Enctype: Tipo de codificacion utilizado para enviar el formulario

Elementos del formulario

input: Es el elemento principal de recolección de datos ya que posee varios tipos de recoleccion

  • text: Presenta una línea de entrada de texto
  • password: Presenta una línea de entrada de texto, lo que el usuario digita se visualiza como un símbolo, generalmente un círculo o un asterisco
  • hidden: permite definir un dato que será enviado al servidor el cual no podrá ser visualizado por el usuario
  • radio: presenta un conjunto de datos de los cuales el usuario podrá escoger solo uno
  • checkbox: Presenta un conjunto de datos y el usuario podrá escoger uno o varios datos
  • file: Presenta un elemento de selección de archivo, el usuario podrá seleccionar uno para ser enviado al servidor, debe ser usado en conjunto con el método POST y la codificación multipart/form-data
  • submit: Presenta un botón que al ser activado iniciará el envío de los datos al servidor
  • reset: Presenta un botón que al ser activado eliminara todos los datos asignados en el formulario
select: presenta una lista de selección desplegable, el usuario podrá seleccionar una opción. Si se utiliza el archivo el atributo multiple en la creación el usuario podrá seleccionar múltiples
textArea: Presenta un recuadro de texto, el usuario podrá asignar varias línea de texto, el número de filas a presentar se controla con el atributo rows y el de columnas con el atributo columns.
label: Asocia una cadena de texto a un elemento del formulario, la asociación se lleva a cabo por el valor del atributo id del elemento
fieldSet: Agrupa varios elementos del formulario en un recuadro de texto al cual puede asignársele un título con la etiqueta legend.

Definicion:
Los estilos permiten modificar la apariencia predeterminada de los elementos html, la modificación se realiza utilizando el sgte formato: propiedad: valor;
Donde:

  • propiedad: indica la característica del elemento html que queremos cambiar
  • valor: el número valor para la característica indicada

    • Definicion:
    Cadena de texto que especifica el elemento html al cual le van a aplicar las definiciones de estilo.
    Existen diferentes tipos de selectores:

    Selector universal(*)
    Especifica todo elemento html
    Selector de etiqueta
    Selector de etiqueta
    Selector de id
    Utiliza el valor del atributo id precedido del carácter “#” para especificar el elemento
    Selector de clase
    Utiliza el valor del atributo class precedido del carácter “.” para especificar el elemento
    Selector descendiente
    Especifica un elemento que se encuentra dentro de otro elemento, tiene el sgte formato.
    selector 1 selector 2
    Selector de hijos
    especifica un elemento que se encuentra definido directamente dentro de otro elemento
    selector 1>selector 2
    Selector adyacente
    Especifica un elemento que se encuentra definido inmediatamente después de otro elemento
    selector 1+selector 2
    Selectores de atributo
    Utilizan el nombre del atributo o el nombre y su valor para seleccionar un elemento html
  • [nombreAtributo]
  • [nombreAtributito = “valorAtributo]

    • Definicion:
    Al desplegar el contenido de un documento html el navegador posiciona los elementos de acuerdo al orden de aparición y al tipo de elemento

  • Elemento en línea: Se muestra a continuación de cualquier otro elemento en línea(imágenes, enlaces, etc)
  • Elemento en bloque: Se muestra separado del resto del contenido, incluyendo un salto de línea antes y después del elemento, algunos incluiran una línea en blanco antes y después

    • Marco de referencia (position)
    1. relative: el marco de ref. sera la posicion predeterminada del elemento
    2. absolute: el marco de ref. sera el ultimo contenedor posicionado, si no lo hubiera será el body
    3. fixed: El marco de ref. sera la ventana del navegador
