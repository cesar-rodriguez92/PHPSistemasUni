Definicion:
Libreria basada en javascript que itiliza los selectores de css para agregar codigo a los elementos html, evitando asi los atributos de eventos javascript

Definicion:
Es un conjunto curado de interacciones de interfaz de usuario, efectos, widgets y temas creados en la parte superior de JQuery y JavaScript. Ya sea que esté creando aplicaciones web altamente interactivas o simplemente necesite agregar un selector de fechas a un control de formulario, jQuery UI es la opción perfecta.

Definicion:
Lenguaje de programacion interpretado cuyo entorno de ejecución es el servidor web, es decir, se ejecutra dentro del servidor web. El propósito principal de php es el de poder generar (html, css, javascript) de manera dinamica, teniendo como fuente u origen del contenido, elementos externos como archivos, directorios, bases de datos, sistemas de correo, snmp, y cualqueire otro archivo.
El motor PHP es enlazado al servidor como modulo o como CGI(Common Gateway Interface), una vez enlazado, podemos usar como nombre de la pagina predeterminada index.php. Los archivos que contengan codigo PHP deben utilizar la extension php.
Cualquier contenido fuera de estos delimitadores será interpretado como código HTML, Esto permite combinar el código PHP con HTML fácilmente; aunque siempre es preferible es preferible separar el código por función: PHP para procesar datos y HTML, JS, CSS solo para la presentación. El código PHP debe estar incluido siempre dentro de los sgtes delimitadores:

<?php
código PHP
?>
